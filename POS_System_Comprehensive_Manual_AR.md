## 🛠️ المميزات الوظيفية الكبرى

### 🛒 نظام البيع وإدارة السلة
- **البحث الذكي**: ابحث بالاسم أو الباركود أو الكود في حقل واحد.
- **التصنيفات الملونة**: وصول سريع للأصناف عبر مجموعات منظمة بصرياً.
- **تعدد طرق الدفع**:
  - **نقدي (Cash)**: مع حساب المتبقي آلياً.
  - **على الحساب (Credit)**: ربط المبيعات بمديونية العملاء.
- **إدارة المرتجعات**: إمكانية إرجاع صنف معين أو الفاتورة بالكامل مع تحديث المخزون والحسابات فوراً.

### ⏳ إدارة الورديات (Sessions)
- **بداية الوردية**: تسجيل الرصيد الافتتاحي وفتح الجلسة باسم الكاشير.
- **نهاية الوردية**: تقرير شامل للمبيعات (نقدي/آجل) ومقارنة العجز والزيادة قبل الإغلاق.
- **أتمتة تقرير الإغلاق**: يقوم النظام فور إغلاق الوردية بتوليد تقرير PDF شامل وحفظه تلقائياً على السيرفر للرجوع إليه في أي وقت.

### 📦 إدارة المخازن والأصناف
- **تعدد الوحدات**: (قطعة، صندوق، كرتونة) مع معاملات تحويل دقيقة.
- **تتبع المخزون**: تحديث لحظي للكميات في جميع الفروع والمخازن.
- **تسوية المخزون**: شاشة مخصصة لضبط الكميات الفعلية وعلاج الفروقات.

## 📊 التقارير الذكية
- **كشف حساب عميل**: تتبع كامل لمدفوعات ومشتريات أي عميل مع تصدير PDF.
- **حالة المخزون**: تقرير بالأصناف التي وصلت لحد الطلب.
- **تقرير المبيعات والارتجاع**: تقارير تفصيلية تدعم الآن ظهور **اسم الكاشير** في كل عملية (بيع أو إرجاع) لضمان دقة الرقابة.
- **تصدير Excel/PDF**: دعم كامل لتصدير جميع التقارير بصيغ احترافية.
